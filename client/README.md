## 📝 todo-list 만들기

### 1주차 미션
#### 구현
- todo list
- CRD
- 브라우저 껐다 켜도 유지되어야 함
- 디자인 자유
- 추가 버튼: 추가 아이콘 svg 넣기
- 삭제 버튼: 삭제 아이콘 png 넣기

#### 조건
- 웹팩 사용 (브라우저 점유율 99%)
- 웹팩 플러그인 사용
- 웹팩 개발 서버 사용 (HMR)
- 바닐라 JS로만 (React X)
- DOM API 적극 활용
- index.html 바디 노드 내부에는 엘리먼트 두개 이상 사용금지
